# practice_vue
# 簽到app   

要求： 
1. 可以註冊用戶 
2. 可以登錄 
3. 點擊簽到按鈕，可以記錄簽到時間，地點。簽到需要限制一天早上，中午，晚上個時間段，分別只允許一次簽到。
4. 用戶，簽到時間等相關數據可以通過【react-native-local-mongodb】等管理。
5. 可以查看簽到的統計結果。
6. 可以對應（Android,iOS）手機，平板。
7. 1到5沒有提到的功能可以自由發揮。
